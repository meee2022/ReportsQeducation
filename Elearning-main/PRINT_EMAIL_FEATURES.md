# خصائص الطباعة والبريد الإلكتروني

تم إضافة خاصيتين جديدتين لصفحة "أداء المعلمين":

## 1️⃣ طباعة التقارير (PDF)

### ✅ طباعة تقرير معلم فردي:
1. اذهب إلى صفحة "أداء المعلمين"
2. اضغط على اسم المعلم لفتح نافذة التفاصيل
3. اضغط زر "طباعة" 🖨️
4. سيتم تنزيل ملف PDF يحتوي على:
   - معلومات المعلم الأساسية
   - إحصائيات الأداء
   - جدول تفصيلي بجميع الصفوف والمواد

### ✅ طباعة تقرير القسم (جميع المعلمين):
1. في صفحة "أداء المعلمين"
2. طبّق الفلاتر المطلوبة (مدرسة، مادة، صف)
3. اضغط زر "طباعة التقرير" أعلى الجدول
4. سيتم تنزيل PDF يحتوي على:
   - إحصائيات عامة
   - جدول بجميع المعلمين حسب الفلاتر

---

## 2️⃣ إرسال البريد الإلكتروني

### ✅ إرسال تقرير معلم بالبريد:
1. افتح تفاصيل المعلم
2. اضغط زر "إرسال" 📧
3. أدخل البريد الإلكتروني المستهدف
4. عدّل الموضوع والرسالة (اختياري)
5. اضغط "إرسال"

### ⚙️ إعداد خادم البريد (مرة واحدة):

**الخطوة 1: تثبيت المتطلبات**
```powershell
cd backend
pip install -r requirements_email.txt
```

**الخطوة 2: إعداد Gmail**
1. تفعيل المصادقة الثنائية (2FA)
2. إنشاء App Password: https://myaccount.google.com/apppasswords
3. إنشاء ملف `.env` في مجلد `backend`:
```env
EMAIL_HOST=smtp.gmail.com
EMAIL_PORT=587
EMAIL_USER=your-email@gmail.com
EMAIL_PASSWORD=your-16-char-app-password
```

**الخطوة 3: تشغيل خادم البريد**
```powershell
cd backend
python email_server.py
```

سيعمل الخادم على http://localhost:5000

## 🛠️ استكشاف الأخطاء

### مشكلة: لا يتم تنزيل PDF
- تأكد من السماح للمتصفح بتنزيل الملفات
- جرّب متصفح آخر

### مشكلة: فشل إرسال البريد
- تأكد من تشغيل `python email_server.py`
- تحقق من إعدادات `.env`
- تأكد من App Password صحيح
- راجع console للمزيد من التفاصيل

## 🎯 الاستخدام المتقدم

1. احصل على ملف خط `.ttf` عربي
2. حوّله إلى base64
3. أضفه في `printTeacherReport.js`
